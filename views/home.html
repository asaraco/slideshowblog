Welcome!

This is Aaron Saraco's website. Currently, it is mostly a testing ground to demonstrate various skills I've been learning. I also plan to use it as a real blog some day, when I feel it's ready for prime time.

The main web technologies used here right now are:

  • HTML
  • CSS (using Sass for preprocessing)
  • Angular.js (and once it's in a good place, I'll see about Angular 2)
  • jQuery
  • Apache HTTPD web server
  • SVG
  • MongoDB - currently has 1 collection for reviews and 1 collection for user IDs/passwords
  • Node.js & Express.js - handles AJAX requests between website and MongoDB

The slideshow you see above (including somewhat odd choices of placeholder images) was the first big part of this project. I figured since pretty much every major website has one of those, I should try my hand at it. Using Chrome with a touchscreen, or a mobile device, it should be touch-sensitive. Originally all of the slides were hard-coded, but they are now loaded in from a JSON file.

The next major thing I worked on was the "music database" page. This is still a work in progress (moreso than everything else) and I plan to separate it out into its own thing eventually. (But then I wouldn't have enough links on the navbar!)

I hope you enjoy this site, for whatever it's worth right now. Try not to break anything!
